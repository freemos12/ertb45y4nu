# Random Task Generator

**Автор:** [Воробьев Семён]

## Описание

Приложение «Random Task Generator» — это GUI‑программа на Python для генерации случайных задач с возможностью сохранения истории и фильтрации.

### Основные функции:
* Генерация случайной задачи из предопределённого списка.
* Добавление пользовательских задач с указанием типа.
* Сохранение истории сгенерированных задач в JSON‑файл.
* Фильтрация истории по типам задач: учёба, спорт, работа.
* Очистка истории.
